## [0.1.0] - 2026-06-26 (начальное состояние)

### Стек

- Next.js 16 + TypeScript + Tailwind CSS
- Prisma ORM + SQLite (для разработки)
- App Router

### Структура проекта

```
src/app/
├── (public)/
│   ├── categories/        — страницы категорий
│   ├── stories/           — страницы сказок
│   └── tags/              — страницы тегов
├── (admin)/               — CMS панель
├── feed.xml/              — RSS лента
├── sitemap.ts             — динамическая карта сайта
├── robots.ts              — директивы для ботов
├── layout.tsx             — корневой layout
└── page.tsx               — главная страница
```

### Модели БД

- `Story` — сказки (title, slug, storyText, audioUrl, coverImageUrl, metadata)
- `StoryImage` — иллюстрации к сказкам
- `StoryParagraph` — абзацы сказок
- `Category` — категории (с вложенностью)
- `Tag` — теги
- `StoryTag` — связь сказок и тегов
- `SeoMetadata` — SEO для каждой сказки

### Ключевые выводы

1. **Побеждает объём** — casino.ru: 726 казино, deti-online.com: 1700+ сказок
2. **Фильтры = SEO** — каждый фильтр создаёт новую индексируемую страницу
3. **Возрастная сегментация** — критично для детского контента
4. **Авторские страницы** — отдельные URL по авторам
5. **Внутренняя перелинковка** — сказка → категория → тег → похожие
